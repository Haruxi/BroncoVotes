BigIntToSolBigInt

Since Solidity cannot handle values larger than 256 bits, the solution is to split huge values into smaller values: convert Javascript's BigInt to binary, split it into arbitrary sizes (default is 256 bits), convert the individual binary numbers that were split into decimal numbers, and The divided individual binary numbers are converted to decimal numbers and stored in an array.

new BigIntToSolBigInt(value: BigInt, bitLength: number, size: number): Array<number>
Parameters
value (BigInt) target number
bitLength (number = 1024) Bit length upper limit
size (number = 256) Split size (bit)
Returns
Array<number>: result - Converted numbers
